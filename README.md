# 🚀 Neon Mission Control

A retro-futuristic productivity dashboard designed to gamify studying and task management.

## 🎯 Features
- Add and complete missions (tasks)
- Progress tracking system
- Level system based on completion
- Focus timer (Pomodoro style)
- Live clock
- AI-based suggestions
- Neon-themed UI

## 🛠️ Technologies Used
- HTML (structure)
- CSS (styling and layout)
- JavaScript (functionality and interactivity)

## ⚙️ How to Use

1. Enter a task in the "Missions" section
2. Click "Add" to create a mission
3. Click on a mission to mark it complete
4. Track your progress and level
5. Use the focus timer to stay productive

## 📌 Concept

This project transforms productivity into a gamified experience by treating tasks as missions and progress as levels, inspired by retro-futuristic aesthetics.
